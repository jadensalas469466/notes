2. 纵向轰炸

## 3 Bypass

```
手机号前加 86/00
添加字符绕过，手机号后加任意字母，如 `x`，或加空格，或 `/n` 
删除或修改 Cookie 值
修改返回值为 0,1,true,success (状态码：200)
双写 (同任意用户漏洞案例双写)
XFF 伪造为内网 IP ：X-Forwarded-For:127.0.0.1 (没有这个参数可以手动添加，一般在倒数第二排，可使用插件 burpfakeip 伪造)
垃圾字符
00 截断
Burp 拦截后狂点发送按钮
```
