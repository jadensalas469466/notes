1. 利用 SMB 协议共享数据，使 NAS 中的数据可在局域网内被访问。
2. 映射 NAS 的 SMB 网络驱动器到计算机。
3. 映射之后两台计算机都可使用 FreeFileSync 双向同步与 NAS 交换数据。
4. NAS 可以使用 Syncthing 单向同步将数据备份到 服务器。
5. 使用 RaiDrive 映射服务器 的 SFTP 网络驱动器到本地，即可随时访问备份数据。
