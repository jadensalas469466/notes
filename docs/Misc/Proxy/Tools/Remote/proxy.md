用于配置全局代理.

## 1. Install

安装

```
curl -fsSL https://github.com/jadensalas469466/scripts/raw/main/proxy_install.sh | bash
```

加载环境变量

```
source ~/.zshrc
```

## 2. Usage

配置代理

```
proxy <ip:port>
```

取消代理

```
proxy
```
