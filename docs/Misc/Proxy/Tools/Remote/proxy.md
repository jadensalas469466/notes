用于配置全局代理.

## 1. Install

安装

```
curl -fsSL https://github.com/jadensalas469466/tools/raw/main/other/proxy_install.sh | bash
```

写入环境变量

```
echo "source ~/.local/bin/proxy.sh" >> ~/.zshrc && source ~/.zshrc
```

## 2. Usage

配置代理

```
proxy <ip:port>
```

取消代理

```
proxy
```
