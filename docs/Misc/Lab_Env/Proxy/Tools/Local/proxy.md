用于配置全局代理.

## 1. Install

安装

```
┌──(nemo@debian)-[~]
└─$ curl -fsSL https://github.com/jadensalas469466/script/raw/main/proxy_install.sh | bash
```

> 需加载环境变量

## 2. Usage

配置代理

```
┌──(nemo@debian)-[~]
└─$ proxy 127.0.0.1:10808
```

取消代理

```
┌──(nemo@debian)-[~]
└─$ proxy
```
