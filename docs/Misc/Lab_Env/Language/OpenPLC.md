 OpenPLC Runtime version 3
 
## 1. Install

安装

```
┌──(nemo@debian)-[~]
└─$ git clone https://github.com/thiagoralves/OpenPLC_v3.git \
&& cd OpenPLC_v3 \
&& ./install.sh linux
```
## 2. Uase

运行

```
┌──(nemo@debian)-[~/OpenPLC_v3]
└─$ sudo ./start_openplc.sh
```
停止

```
┌──(nemo@debian)-[~/OpenPLC_v3]
└─$ sudo ./stop_openplc.sh
```
