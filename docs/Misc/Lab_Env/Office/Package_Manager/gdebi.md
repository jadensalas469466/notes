gdebi is a simple to tool install local deb files or install the
build-dependencies of .dsc files.

## 1.Install

安装

```
┌──(nemo@debian)-[~]
└─$ sudo apt install -y gdebi-core
```

## 2.Usage

使用

```
┌──(nemo@debian)-[~]
└─$ sudo gdebi -n app.deb
```
