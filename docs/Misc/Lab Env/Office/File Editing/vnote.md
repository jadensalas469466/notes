A pleasant note-taking platform in native C++.

## 1. Install

安装

```
┌──(nemo@debian)-[~]
└─$ curl -LO https://github.com/vnotex/vnote/releases/download/v3.17.0/vnote-linux-x64_v3.17.0.zip \
&& unzip ./vnote*.zip && rm ./vnote*.zip
```

双击运行 vnote.AppImage 将应用集成到系统菜单中
