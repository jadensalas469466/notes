## WePE

插入 U 盘，运行 WePE

选择安装 PE 到 U 盘

安装方法

```
方案一：UEFI/Legacy全能三分区方式(推荐)
```

待写入 U 盘

```
U 盘
```

格式化

```
exFAT USB-HDD
```

U 盘卷标

```
SD
```

立即安装 PE 到 U 盘

开始制作 

完成安装 

将 Windows.ios 移动到 PE 的 U 盘分区

重启系统进入引导界面

```
需要关闭 Bitlocker
ASUS 是在启动引导界面按 Del 
TinkPad 是在启动引导界面按 F12 
```

选择 U 盘启动进入 WePE

进入 `Win.iso` ，运行 `setup.exe` ，进行安装
