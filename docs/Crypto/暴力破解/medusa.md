```
Medusa v2.2 [http://www.foofus.net] (C) JoMo-Kun / Foofus Networks <jmk@foofus.net>

medusa: 选项需要一个参数 -- 'h'
CRITICAL: 处理命令行选项时发生未知错误。
ALERT: 必须提供主机信息。

语法: Medusa [-h 主机名|-H 文件] [-u 用户名|-U 文件] [-p 密码|-P 文件] [-C 文件] -M 模块 [选项]
  -h [文本]    : 目标主机名或 IP 地址
  -H [文件]    : 包含目标主机名或 IP 地址的文件
  -u [文本]    : 要测试的用户名
  -U [文件]    : 包含要测试的用户名的文件
  -p [文本]    : 要测试的密码
  -P [文件]    : 包含要测试的密码的文件
  -C [文件]    : 包含组合条目的文件。更多信息请参阅 README。
  -O [文件]    : 要附加日志信息的文件
  -e [n/s/ns]  : 额外的密码检查（[n] 无密码，[s] 密码=用户名）
  -M [文本]    : 要执行的模块的名称（不带 .mod 扩展名）
  -m [文本]    : 要传递给模块的参数。可以多次传递此参数，每次使用不同的参数（例如
                 -m 参数1 -m 参数2，等等）
  -d           : 显示所有已知模块
  -n [数值]    : 用于非默认 TCP 端口号
  -s           : 启用 SSL
  -g [数值]    : 在尝试连接 NUM 秒后放弃（默认为 3 秒）
  -r [数值]    : 在重试尝试之间睡眠 NUM 秒（默认为 3 秒）
  -R [数值]    : 在放弃之前尝试 NUM 次重试。总尝试次数将为 NUM + 1。
  -c [数值]    : 等待使用微秒验证套接字是否可用的时间（默认为 500 微秒）。
  -t [数值]    : 同时测试的登录总数
  -T [数值]    : 同时测试的主机总数
  -L           : 使用每个线程一个用户名并行化登录。默认情况下是在继续之前处理整个用户名。
  -f           : 找到第一个有效的用户名/密码后停止扫描主机。
  -F           : 在任何主机上找到第一个有效的用户名/密码后停止审核。
  -b           : 抑制启动横幅
  -q           : 显示模块的使用信息
  -v [数值]    : 详细级别 [0 - 6（更多）]
  -w [数值]    : 错误调试级别 [0 - 10（更多）]
  -V           : 显示版本
  -Z [文本]    : 根据先前扫描的映射恢复扫描
  
```
