Identify the technology stack that powers a website and explore the Web of Things.

## 1. install

安装

```
apt install -y whatweb
```

## 2. use

识别目标使用的 Web 服务

```
whatweb -a 3 -v --color=never https://example.com > whatweb_example.com.txt
```
