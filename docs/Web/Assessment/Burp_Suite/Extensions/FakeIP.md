一款用于伪造 IP 地址的 Burp Suite 插件。

## 1 使用

伪造 IP

```
customip	伪造指定 IP
127.0.0.1	伪造本地 IP
randomip	伪造随机 IP
autoxff		伪造随机 IP 爆破
```
