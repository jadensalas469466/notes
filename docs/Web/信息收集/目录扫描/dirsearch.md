目录扫描。

## 1 安装

安装

```shell
┌──(root㉿kali)-[~]
└─# sudo apt install -y dirsearch
```

## 2 使用

查看帮助

```shell
┌──(root㉿kali)-[~]
└─# man dirsearch
```

对目标进行目录扫描

```shell
┌──(root㉿kali)-[~]
└─# dirsearch -u [url]
```
