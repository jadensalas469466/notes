网络爬虫。

# 1 部署

安装

```shell
┌──(root㉿kali)-[~]
└─# go install github.com/projectdiscovery/katana/cmd/katana@latest
```

# 2 使用

查看帮助

```shell
┌──(root㉿kali)-[~]
└─# katana -h
```

对目标进行爬虫

```shell
┌──(root㉿kali)-[~]
└─# katana -u [url]
```
