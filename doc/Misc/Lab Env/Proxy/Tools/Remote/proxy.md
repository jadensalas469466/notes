用于配置全局代理.

## 1. Install

安装

```
┌──(nemo@infosec)-[~]
└─$ curl -fsSL https://github.com/jadensalas469466/script/raw/main/proxy_install.sh | bash
```

加载环境变量

```
┌──(nemo@infosec)-[~]
└─$ source ~/.zshrc
```

## 2. Usage

配置代理

```
┌──(nemo@infosec)-[~]
└─$ proxy <ip:port>
```

取消代理

```
┌──(nemo@infosec)-[~]
└─$ proxy
```
